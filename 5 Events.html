Explanation of the Example

  1. HTML Structure: A parent <div> containing a child <div>.
  2. Event Listeners:
    1. Capture Phase: The parent listener will execute first when the event is captured (true flag).
    2. Target Phase: If you click on the child element, the child’s event listener is executed next, displaying its alert.
    3. Bubble Phase: If you do not stop propagation in the child’s event handler, the parent’s listener will execute after the child’s listener, showing its alert.

Summary

  1. Event Propagation: Refers to the way events are handled in a DOM tree structure.
  2. Phases: Events go through three phases: Capture (down), Target (at the element), and Bubble (up).
  3. Event Listeners: Can be set to trigger during capture, target, or bubble phases, allowing for flexible event handling.
